# AI-Powered News Sentiment Analysis Pipeline

## Project Overview
An AI-powered Python pipeline that fetches real-time technology 
news, analyzes sentiment using NLP, and generates an automated 
analysis report.

## Features
- Fetches live tech news from REST API
- AI sentiment analysis (Positive/Negative/Neutral)
- Automated report generation
- NLP powered by TextBlob

## 🛠️ Technologies Used
- Python
- TextBlob (NLP/AI)
- REST API
- Google Colab

## 📊 Pipeline Flow
Fetch News (API) → Analyze Sentiment (AI/NLP) → Generate Report

## 📈 Sample Output
- 🟢 Positive News: 10%
- 🔴 Negative News: 10%
- 🟡 Neutral News: 80%
- 💡 Overall Sentiment: Neutral

## ▶️ How to Run
1. Open .ipynb file in Google Colab
2. Run all cells in order
3. Report generates automatically
